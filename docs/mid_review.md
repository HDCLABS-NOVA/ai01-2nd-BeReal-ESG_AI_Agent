## 📌 3. 주요 개발 항목별 상세 점검

### ✅ 3.1 데이터 처리 파이프라인

- PDF → Text 추출(PyPDF, Tesseract)  
- Text Chunking 적용  
- 벡터 임베딩(bge-m3) 정상 생성  
- ChromaDB에 저장 완료  

**상태:** 정상  
**향후 작업:** 데이터 정합성 점검 자동화 필요

---

### 🧠 3.2 RAG 검색 품질 점검

- 정책 문서 검색 정확도: 약 85%  
- 잘 검색되는 분야: 환경(E), 안전(S)  
- 부족한 분야: GRI 사회적 지표 일부  

**상태:** 양호  
**개선 필요:** 문서별 가중치 조정(MMR)

---

### 📑 3.3 정책 분석 모듈

| 항목 | 상태 | 코멘트 |
|------|--------|-----------|
| 요약 기능 | ✔ 정상 동작 | 결과 일관성 양호 |
| 비교 기능 | ▲ 일부 미흡 | 용어 매핑 고도화 필요 |

**이슈:**
